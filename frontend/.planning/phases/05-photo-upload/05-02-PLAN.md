  <name>Task 1: Create CropPhotoModal component with tests</name>
  <files>src/components/profile/CropPhotoModal.tsx, src/__tests__/CropPhotoModal.test.tsx</files>
  <action>
    Create `src/components/profile/CropPhotoModal.tsx`:

    **Props interface:**
    ```typescript
    interface CropPhotoModalProps {
      open: boolean;
      imageUrl: string;        // Object URL from file picker
      onClose: () => void;     // Closes modal and revokes object URL
      onUploaded: () => void;  // Called after successful upload (for parent cleanup)
    }
    ```

    **Component structure:**
    - Use `AppDialog` as the modal wrapper with `open` and `onClose` props
    - `disableEscapeKeyDown` on AppDialog to prevent accidental dismissal during crop (same pattern as EditNameModal)
    - `DialogTitle`: "Recortar foto"
    - `DialogContent`:
      - `Box` with `position: relative`, `height: 350px`, `width: 100%`, `bgcolor: 'black'` — this is the Cropper container
      - Inside: `<Cropper image={imageUrl} crop={crop} zoom={zoom} aspect={3/4} cropShape="rect" showGrid={true} onCropChange={setCrop} onZoomChange={setZoom} onCropComplete={(_, area) => setCroppedAreaPixels(area)} />`
      - Below cropper container: `Box` with `px: 2, pt: 2` containing MUI `Slider` for zoom control with `min={1}`, `max={3}`, `step={0.01}`, `value={zoom}`, `onChange={(_, v) => setZoom(v as number)}`
    - `DialogActions`:
      - `Button onClick={onClose}`: "Cancelar"
      - `LoadingButton loading={mutation.isPending} onClick={handleSave}`: "Salvar"

    **handleSave logic:**
    1. If `!croppedAreaPixels` return early
    2. `const blob = await getCroppedImg(imageUrl, croppedAreaPixels)`
    3. `const file = new File([blob], 'photo.jpg', { type: 'image/jpeg' })`
    4. `mutation.mutate(file, { onSuccess: () => { enqueueSnackbar('Foto atualizada!', { variant: 'success' }); onClose(); onUploaded(); }, onError: () => { enqueueSnackbar('Erro ao enviar foto.', { variant: 'error' }); } })`

    **State:** `crop: { x: 0, y: 0 }`, `zoom: 1`, `croppedAreaPixels: Area | null`

    **Cleanup:** `useEffect` that revokes `imageUrl` when component unmounts or `imageUrl` changes: `return () => URL.revokeObjectURL(imageUrl)`

    **Styles:** Semi-transparent dark overlay is handled by react-easy-crop by default. Rectangular crop shape is set via `cropShape="rect"`. The black background on the container provides contrast.

    Create `src/__tests__/CropPhotoModal.test.tsx`:
    - Mock `react-easy-crop` default export as a div that renders with data-testid="cropper"
    - Mock `../utils/cropImage` and `../hooks/useUploadPhoto`
    - Test 1: "renders crop modal with Cropper when open" — verify dialog title "Recortar foto", cropper element, zoom slider, Salvar/Cancelar buttons are present
    - Test 2: "renders zoom slider with correct range" — verify slider exists with min=1, max=3
    - Test 3: "calls onClose when Cancelar is clicked"
  </action>
