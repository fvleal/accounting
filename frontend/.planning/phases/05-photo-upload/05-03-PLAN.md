**Rewrite CropPhotoModal to use react-image-crop:**

Replace the entire CropPhotoModal implementation. Key changes:

1. **Import react-image-crop** instead of react-easy-crop:
   ```typescript
   import ReactCrop from 'react-image-crop';
   import type { Crop, PixelCrop } from 'react-image-crop';
   import 'react-image-crop/dist/ReactCrop.css';
   ```

2. **State**: Replace `crop: {x,y}`, `zoom`, and `croppedAreaPixels` with a single `crop` state of type `Crop`. Initialize with a centered 3x4 crop:
   ```typescript

   const [completedCrop, setCompletedCrop] = useState<PixelCrop>();
   const imgRef = useRef<HTMLImageElement>(null);
   ```

3. **Remove the useEffect that revokes object URL** -- this caused the black background bug in Strict Mode. Object URL revocation moves to ProfilePage.

   ```tsx

     crop={crop}
     onChange={(c) => setCrop(c)}

     <img
       ref={imgRef}
   ```

   ```css
   .ReactCrop__overlay {
   ```

   Add these CSS rules to `src/index.css` in the `@layer components` section (following project CSS layer convention from Phase 01).
