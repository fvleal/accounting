  <action>
    Build all 4 queries using TDD (RED then GREEN for each).

    **GetAccountByIdQuery:**
    - Input: `{ id: string }`
    - Output: `{ id, auth0Sub, name, email, cpf, birthDate, phone, photoUrl, createdAt, updatedAt }`
    - execute(): findById -> throw AccountNotFoundError if null -> toOutput

    **GetMeQuery** (dedicated use case per CONTEXT.md, NOT reusing FindAccountByFieldQuery):
    - Input: `{ auth0Sub: string }`
    - Output: same shape as GetAccountByIdOutput
    - execute(): findByAuth0Sub -> throw AccountNotFoundError if null -> toOutput

    **FindAccountByFieldQuery:**
    - Input: `{ field: 'email' | 'cpf', value: string }`
    - Output: same shape as GetAccountByIdOutput
    - execute(): switch on field, call findByEmail or findByCpf -> throw AccountNotFoundError if null -> toOutput
    - Follow exact pattern from RESEARCH.md (Pattern: FindAccountByFieldQuery)

    **ListAccountsQuery:**
    - Input: `{ limit: number, offset: number }`
    - Output: `{ data: AccountSummary[], total: number }` where AccountSummary = `{ id, auth0Sub, name, email, cpf, birthDate, phone, photoUrl, createdAt, updatedAt }`
    - execute(): call findAll(params) -> map data to output

    All queries: @Injectable(), @Inject(ACCOUNT_REPOSITORY_PORT), implement UseCase<I, O>.

    Test mocking: create mock AccountRepositoryPort with vi.fn(). Use Account.create() for fixtures (requires auth0Sub).

    ```typescript
    import { Module } from '@nestjs/common';
    import { AccountInfrastructureModule } from '../infrastructure/account-infrastructure.module';
    // Import all 5 commands
    import { CreateAccountCommand } from './commands/create-account.command';
    import { UpdateNameCommand } from './commands/update-name.command';
    import { UpdatePhoneCommand } from './commands/update-phone.command';
    import { UpdateBirthDateCommand } from './commands/update-birth-date.command';
    import { UploadAccountPhotoCommand } from './commands/upload-account-photo.command';
    // Import all 4 queries
    import { GetAccountByIdQuery } from './queries/get-account-by-id.query';
    import { GetMeQuery } from './queries/get-me.query';
    import { FindAccountByFieldQuery } from './queries/find-account-by-field.query';
    import { ListAccountsQuery } from './queries/list-accounts.query';

    @Module({
      imports: [AccountInfrastructureModule],
      providers: [
        CreateAccountCommand,
        UpdateNameCommand,
        UpdatePhoneCommand,
        UpdateBirthDateCommand,
        UploadAccountPhotoCommand,
        GetAccountByIdQuery,
        GetMeQuery,
        FindAccountByFieldQuery,
        ListAccountsQuery,
      ],
      exports: [
        CreateAccountCommand,
        UpdateNameCommand,
        UpdatePhoneCommand,
        UpdateBirthDateCommand,
        UploadAccountPhotoCommand,
        GetAccountByIdQuery,
        GetMeQuery,
        FindAccountByFieldQuery,
        ListAccountsQuery,
      ],
    })
    export class AccountApplicationModule {}
    ```
