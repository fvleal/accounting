<task type="auto">
  <name>Task 2: Create JwtStrategy, JwtAuthGuard, RolesGuard, and AuthModule with APP_GUARD</name>
  <files>
    src/shared/infrastructure/auth/strategies/jwt.strategy.ts,
    src/shared/infrastructure/auth/guards/jwt-auth.guard.ts,
    src/shared/infrastructure/auth/guards/roles.guard.ts,
    src/shared/infrastructure/auth/auth.module.ts
  </files>
  <action>
    1. Create `src/shared/infrastructure/auth/strategies/jwt.strategy.ts`:
       - @Injectable() class JwtStrategy extends PassportStrategy(Strategy)
       - Constructor injects ConfigService
       - super() config: secretOrKeyProvider from passportJwtSecret with cache=true, rateLimit=true, jwksRequestsPerMinute=5, jwksUri from AUTH0_DOMAIN env var
       - jwtFromRequest: ExtractJwt.fromAuthHeaderAsBearerToken()
       - audience from AUTH0_AUDIENCE env var
       - issuer: `https://${AUTH0_DOMAIN}/`
       - algorithms: ['RS256']
       - validate(payload: JwtPayload): returns payload directly (Passport attaches to request.user)

    2. Create `src/shared/infrastructure/auth/guards/jwt-auth.guard.ts`:
       - @Injectable() class JwtAuthGuard extends AuthGuard('jwt')
       - Constructor injects Reflector
       - Override canActivate: check IS_PUBLIC_KEY metadata via reflector.getAllAndOverride on handler and class. If isPublic=true, return true. Otherwise call super.canActivate(context)

    3. Create `src/shared/infrastructure/auth/guards/roles.guard.ts`:
       - @Injectable() class RolesGuard implements CanActivate
       - Constructor injects Reflector
       - canActivate: get requiredRoles from ROLES_KEY metadata via reflector.getAllAndOverride. If no roles required, return true. Extract user from request. Get permissions array from user.permissions (default to empty array). Return true if any requiredRole is found in permissions array.
       - IMPORTANT: Check user.permissions (Auth0 RBAC populates permissions claim, NOT roles claim per RESEARCH.md Pitfall 7)

    4. Create `src/shared/infrastructure/auth/auth.module.ts`:
       - @Module with imports: [PassportModule.register({ defaultStrategy: 'jwt' })]
       - providers: JwtStrategy, { provide: APP_GUARD, useClass: JwtAuthGuard }, { provide: APP_GUARD, useClass: RolesGuard }
       - IMPORTANT: JwtAuthGuard MUST be registered BEFORE RolesGuard in providers array (Pitfall 3 -- guard execution order)
       - exports: [PassportModule]

    5. Update barrel export in index.ts to include AuthModule, JwtAuthGuard, RolesGuard.
  </action>
