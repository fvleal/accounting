    1. Create `src/account/interface/filters/domain-exception.filter.ts`:
       - @Catch() decorator (catches all exceptions)
       - Implements ExceptionFilter
       - Private static readonly CODE_TO_STATUS map: ACCOUNT_NOT_FOUND->404, DUPLICATE_EMAIL->409, DUPLICATE_CPF->409, DUPLICATE_AUTH0_SUB->409
       - catch(exception, host) method:
         a. If exception instanceof DomainException: map code to status (default 422 if code not in map), respond with { statusCode, error: exception.code, message: exception.message, details: exception.metadata }
         b. If exception instanceof HttpException: get status and response. For UnprocessableEntityException (422 from ValidationPipe): respond with { statusCode: 422, error: 'VALIDATION_ERROR', message: 'Validation failed', details: exception.getResponse() } -- the getResponse() will contain the field error array from the custom exceptionFactory
         c. For other HttpExceptions: respond with { statusCode, error: exception.name, message: exception.message, details: null }
         d. For unknown errors: log to console.error, respond with { statusCode: 500, error: 'INTERNAL_ERROR', message: 'Internal server error', details: null }
       - Import DomainException from src/shared/domain/exceptions/domain-exception.base

    2. Create `src/account/interface/interceptors/response-envelope.interceptor.ts`:
       - @Injectable() implements NestInterceptor
       - intercept(context, next): returns next.handle().pipe(map(data => ...))
       - Envelope logic:
         a. If data has shape { data: any[], total: number } (list response from controller): wrap as { data: data.data, meta: { total: data.total, offset: data.offset, limit: data.limit, timestamp: new Date().toISOString() } }
         b. Otherwise (single resource): wrap as { data, meta: { timestamp: new Date().toISOString() } }
       - Import map from rxjs/operators
       - NOTE: Exception filter responses bypass the interceptor (NestJS lifecycle: interceptor map() only applies to successful responses from next.handle())
  </action>
