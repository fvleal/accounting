### 1 Informational Finding

**Domain events emitted with no consumers:** `AccountCreated` and `AccountUpdated` are dispatched via `EventEmitter2` in `PrismaAccountRepository.save()` but no `@OnEvent` listener exists. The event infrastructure is ready for future consumers (audit trail, notifications) but currently fires into void. Not a runtime error — NestJS silently drops unhandled events.
